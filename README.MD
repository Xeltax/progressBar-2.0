#  How to Use
 To use the progress bar  :

 ```lua
 exports['progressBars']:startUI(TimeInMS, TextOnProgressBar, ColorOfProgressBar)
 ```
 
 Example :
 ```lua
 exports['progressBars']:startUI(10000, "Achat en cours", "rgba(0, 16, 194, 0.95)")
 ```
